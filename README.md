# FluB_NAI_CoP
Analysis used for manuscript "Protective effects of influenza B neuraminidase antibodies against symptomatic influenza virus infection". The paper looks at neuraminidase-inhibiting antibodies as a correlate of protection against influenza B among healthcare personnel enrolled in the Study of Healthcare Personnel with Influenza and other Respiratory Viruses in Israel (SHIRI) study. All data produced in the present study are available upon reasonable request to the corresponding author.

## Script Details:
'SHIRI_aim3_clean.R' contains data cleaning, descriptive statistics, and primary analysis code.
'FluB COP CDC Code_LG.R' contains analysis of neuraminidase-inhibiting antibody reduction in hazard against influenza B.

### RStudio Version Details:
